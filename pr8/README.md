Відповідь:
Тут ми програмно створюємо тимчасовий файл, та записуємо у нього дані відповідно до умови завдання, далі за допомогою команди:
```bash
lseek(fd, 3, SEEK_SET);
```
Переходимо у цей файловий дескриптор(fd), пересуваємо вказівник для читання на 3 індекси вперед, та вказуємо рахувати спочатку файлу(SEEK_SET), далі за допомогою команди:
```bash
read(fd, &buffer, 4);
```
читаємо з цього дескриптора 4 байти та записуємо їх до раніше створеного буферу, та виконуємо вивід на екран.

Вивід в консоль:
```bash
./task2

Bites red: 4
What we have in buffer: 2 3 3 7
```
